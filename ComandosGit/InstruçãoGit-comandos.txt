Comandos:

1 - Clonar repositorio:
	git clone <url>

* Enviar alterações 

	2 - Verificar os arquivos que possuem modificação:
		git status
			OBS: Em vermelho esta fora do backstage, em verde está
		    	 Dentro Do backStage 	
	3 - Adicionar todos os arquivo no backStage:
		git add .

	4 - Criar commit para enviar ao repositorio:
		git commit -m <nome do commit>

	5 - Enviar commit ao repositorio:
		git push

	Atualizar as pasta local com o conteudo do repositorio do git:
	
	6- git pull
